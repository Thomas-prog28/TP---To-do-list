Fonctionnalités implémentées :
-bandeau consentement cookie
-gestion de tâches : ajout d'une tâche par remplissage du champ de saisie et clique sur le bouton d'ajout pour mettre dans la liste
-coche ou décoche d'une case (correspondant à chaqque tâche individuellemnet) si la tâche a été réalisé ou pas
-bouton supprimer pour suppression de la tâche s'il a été effectué
-sauvegarde des données dans le localstorge pour persistance si rafraichissement de la page ou redémarrage du navigateur, à chaque ajout, changement d'état ou suppression de données.
-fonctions de filtrages des tâches dans la liste (Toutes / A faire / faites) dynamiquement
-Bouton du filtre actif visible par une surbrillance

les différentes contraintes design responsive implémentées : 
-CSS écrit en mobile first.
-Trois breakpoints minimum : mobile (≤ 600px), tablette (601–1024px), desktop (≥ 1025px).
-Aucun scroll horizontal entre 360px et 1920px.
-Zones cliquables d'au moins 44×44px.
-Taille de police minimale 16px sur mobile.
-Utilisation de Flexbox ou Grid pour la mise en page.
